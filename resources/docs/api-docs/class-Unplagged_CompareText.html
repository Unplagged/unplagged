Class Unplagged_CompareText

This class can be used to find similarities in text corpora.

It is based on the work done by VroniPlag user "Marcusb". The original javascript sources can be found under http://de.vroniplag.wikia.com/wiki/MediaWiki:Fragcolors/code.js(last accessed Nov. 9th 2012).
